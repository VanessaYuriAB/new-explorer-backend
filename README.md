# 🗞️ News Explorer — Backend

API **back-end** do projeto **News Explorer**, responsável por autenticação, gerenciamento
de usuários e persistência de artigos salvos.

Este repositório implementa uma **API REST** robusta, segura e desacoplada, com foco em
**boas práticas de backend**, **segurança** e **testabilidade**.

> 📌 _Repositório do front-end: https://github.com/VanessaYuriAB/news-explorer-frontend_

---

## 🌐 API Online

Base URL: https://api.newsexplorer.sevencomets.com

---

## 🧠 Destaques Técnicos

- Node.js + Express
- MongoDB com schemas bem definidos
- Autenticação e autorização via **JWT**
- Rotas protegidas e separação clara de responsabilidades
- Validação rigorosa de dados
- Tratamento centralizado de erros
- Logs de requisição e erro
- Testes de integração com **Jest** e **Supertest**
- Deploy em servidor próprio na **Google Cloud**

---

## 🧱 Stack

- **Runtime:** Node.js
- **Framework:** Express
- **Banco de dados:** MongoDB
- **Auth:** JWT
- **Testes:** Jest, Supertest
- **Deploy:** Google Cloud (VM)
